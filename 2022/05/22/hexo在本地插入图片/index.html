hexo插入本地图片

2022.05.22 小小的技术

hexo添加本地图片的解决方案

yi. 把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true

er. 在你的hexo目录下执行这样一句话 npm install hexo-asset-image --save，这是下载安装一个HEXO可以上传本地图片的插件hexo-asset-image

san. 等待一小段时间后，再运行hexo n “xxxx”来生成md博文时，/source/_posts文件夹内除了xxxx.md文件还有一个同名的文件夹

注意： xxxx是这个md文件的名字，也是同名文件夹的名字。只需要有文件夹名字即可，不需要有什么绝对路径。你想引入的图片就只需要放入xxxx这个文件夹内就好了，很像引用相对路径。

si 最后在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，然后只需要在xxxx.md中按照markdown的格式引入图片：

![](xxxx/图片名.jpg)

wu. 最后检查一下，hexo g生成页面后，进入public\2022\05\26\index.html文件中查看相关字段，可以发现，html标签内的语句是<img src="2022/05/26/xxxx/图片名.jpg">，而不是<img src="xxxx/图片名.jpg>。这很重要，关乎你的网页是否可以真正加载你想插入的图片。
